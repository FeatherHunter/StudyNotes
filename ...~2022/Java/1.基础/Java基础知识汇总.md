## 多态
1、什么是多态？
> 1-一个引用（类型）在不同情况的多种状态
```java
Cat cat = new Cat();
Dog dog = new Dog();
Animal a = cat;
a.cry(); //Animal 必须有 cry
```
> 2-如果没有cry则使用父类cry
> 3-父类在一定情况下可以转换为子类
