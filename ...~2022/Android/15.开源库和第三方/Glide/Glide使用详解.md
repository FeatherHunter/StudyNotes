# Glide使用详解

版本:2018/9/3-1

## 集成

1、build.gradle
```xml
repositories {
  mavenCentral()
  google()
}
```
```xml
dependencies {
  implementation 'com.github.bumptech.glide:glide:4.8.0'
  annotationProcessor 'com.github.bumptech.glide:compiler:4.8.0'
}
```
